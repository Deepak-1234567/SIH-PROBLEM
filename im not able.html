I'm not able to upload my resume. What do I do?

If you are stuck on the 'Upload your Resume' page and your resume does not upload, it is possible that your browser's plugins are interfering with the upload process. 

Here are a few things you can try:
  1. Click the upload button instead of drag-and-dropping the file into the upload box. If you initially tried clicking the upload button, then try dragging-and-dropping your resume into the upload box instead.
  2. If you're using Chrome, use Incognito mode and try logging in again. Incognito mode clears cookies and disables plugins that might interfere with the logins. You can also use Private Mode in Mozilla Firefox. If you're using any other browser, we recommend using Chrome. 

If you get an error message and your resume failed to process, please see this related article.

For anything else, use the contact form below and we'll get back to you as soon as possible (usually <24 hours)!
